# Creative Card 模板预览

```
┌─────────────────────────────────────────┐
│                                         │
│    ┌─────────────────────────────┐      │
│    │                             │      │
│    │        [头像]               │      │
│    │        用户名               │      │
│    │        简介文字            │      │
│    │   (悬停翻转查看背面)        │      │
│    └─────────────────────────────┘      │
│                                         │
│  ┌───────────────┐  ┌───────────────┐   │
│  │   ┌───────┐   │  │   ┌───────┐   │   │
│  │   │ 图片  │   │  │   │ 图片  │   │   │
│  │   └───────┘   │  │   └───────┘   │   │
│  │   标题文字    │  │   标题文字    │   │
│  │   描述内容... │  │   描述内容... │   │
│  │   [ReadMore] │  │   [ReadMore] │   │
│  └───────────────┘  └───────────────┘   │
│                                         │
│    浮动圆形装饰 (动画背景)               │
│                                         │
└─────────────────────────────────────────┘
```

特点:
- 渐变紫蓝色背景
- 浮动装饰圆形动画
- 3D翻转个人卡片
- 卡片悬停上浮效果
- 图片悬停放大
- 顶部进度条动画
- 入场顺序动画
