# Vintage Style 模板预览

```
┌───────────────────────────────────────┐
│          ╭───────────╮               │
│          │    头像    │               │
│          │  (圆形)   │               │
│          ╰───────────╯               │
│                                       │
│        用户名 (大写字母)               │
│        用户简介 (斜体)                 │
│          ❧ ❦ ❧                       │
├───────────────────────────────────────┤
│                                       │
│   标题 (带下划线)                      │
│                                       │
│   正文内容，优雅的衬线字体，             │

│                                       │
│   ┌─────────────────────┐             │
│   │   装饰图片 (倾斜)    │             │

│                                       │
│   引用文字区域...                      │
│                                       │
│           ✦ 致谢 ✦                    │

```

特点:
- 暖色调复古配色
- 纸张纹理背景
- 衬线字体排版
- 图片轻微倾斜装饰
- 打字机效果标题
- 装饰性分隔符
