# Requirements: Vibe Onepage

**Defined:** 2026-03-22
**Core Value:** Users can have a beautiful, personalized website live in minutes by combining AI-assisted generation with an intuitive block-level editor.

## v1.6 Requirements

### UI Polish

- [ ] **UIP-01**: All pages use white/light background theme with clean, minimal aesthetic
- [ ] **UIP-02**: Components use blue-black color scheme (navy/dark blue accents) for premium feel
- [ ] **UIP-03**: Editor has improved block handles, smoother drag animations, and refined config panel
- [ ] **UIP-04**: All buttons have consistent hover animations (scale, shadow, color transitions)
- [ ] **UIP-05**: Template gallery has polished cards with subtle shadows and refined hover states
- [ ] **UIP-06**: Analytics dashboard matches the new white theme with clean chart styling

## Out of Scope

| Feature | Reason |
|---------|--------|
| Mobile app | Web-only for now |
| Custom domain | Deferred to future |
| Dark mode toggle | Light theme is the focus |
