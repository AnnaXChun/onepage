# Requirements: Vibe Onepage

**Defined:** 2026-03-22
**Core Value:** Users can have a beautiful, personalized website live in minutes by combining AI-assisted generation with an intuitive block-level editor.

## v1.5 Requirements

### Analytics

- [ ] **ANLT-01**: User can view page view trends over time displayed as a line chart with 7/30/90 day range selection
- [ ] **ANLT-02**: User can see referral sources grouped and counted (Google, Bing, direct, social)

## v1.4 Requirements

### Email Collection

- [x] **EML-01**: User must provide a valid email address during registration
- [x] **EML-02**: User can add or update their email address in account settings

### Notifications

- [x] **EML-03**: User receives an email notification when AI website generation is complete
- [x] **EML-04**: User receives an email when their published site receives its first visitor (optional onboarding)

### PDF Delivery

- [x] **EML-05**: User can request the generated PDF to be sent to their registered email
- [x] **EML-06**: PDF email includes the site name and a download link valid for 24 hours

## Out of Scope

| Feature | Reason |
|---------|--------|
| Device/browser breakdown | Nice-to-have, defer to v1.6 |
| Geographic data | Nice-to-have, defer to v1.6 |
| Real-time analytics | Daily aggregates sufficient for v1.5 |
| Analytics email reports | Nice-to-have, defer to future |
| CSV export | Nice-to-have, defer to future |
| General newsletter emails | Not core to website builder value |
| Email marketing automation | Separate product category |
| SMTP configuration by users | Adds complexity, use platform defaults |
