## Milestone Summary

### Key Decisions

- **Resume saved edits**: When opening an existing blog in editor, load from saved blog blocks (not localStorage or template defaults)
- **Status field on blogs table**: Add `status` column with values 'draft'/'published' to distinguish draft from published blogs
- **Separate Drafts section**: On profile page, show distinct "My Drafts" section above/below published sites
- **Continuous auto-save**: Save to backend on changes with 500ms debounce

### Issues Resolved

- EDIT-02 root cause: Editor.tsx loaded from template blocks.json instead of saved blog blocks
- EDIT-03 root cause: handleDone navigated immediately without waiting for debounced save
- DRAFT-03 gap: useAutoSave needed correct blogId to save to correct blog
