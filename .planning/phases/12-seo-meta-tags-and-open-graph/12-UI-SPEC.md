## New Components

### 1. SEO Settings Panel (Slide-over from Toolbar)

**Location:** Editor toolbar, triggered by SEO icon button. Panel slides in from right, overlaying the canvas (does not replace BlockConfigPanel).

**Dimensions:** `w-80` (320px) slide-over panel, `h-full`, positioned `fixed` right-0, top-0.

**Structure:**
```
Header: "SEO Settings" + close button
Tab navigation: [ "Meta Tags" | "robots.txt" ]
---
Meta Tags tab:
  - Meta Title input (max 60 chars)
  - Character counter: "42 / 60"
  - Meta Description textarea (max 160 chars)
  - Character counter: "120 / 160"
  - Info notice (amber): "Re-publish your site for SEO changes to take effect."
  - OG Preview card (shows how link will appear on social)
    - Site title
    - Site description
    - Cover image thumbnail (or placeholder)
    - Badge: "OG" and "Twitter Card"
---
robots.txt tab:
  - Textarea (monospace, ~12 lines visible)
  - Syntax guidance text below textarea
  - Save button
  - Reset to default button
```

**States:**
- Default: Panel hidden (not rendered)
- Open: Panel slides in with `transform: translateX(0)`, 300ms ease-out-quart
- Closing: Panel slides out `transform: translateX(100%)`, 250ms ease-out-quart
- Saving: Save button shows spinner, disabled
- Saved: Brief green checkmark flash, then revert to default

**Inputs:**
- Meta title: `input` with `maxLength={60}`, show live character count
- Meta description: `textarea` with `maxLength={160}`, 3 rows min, show live character count
- robots.txt: `textarea` with `font-mono`, 12 rows, pre-populated with default content

**Info Notice (amber warning):**
```
bg-amber-500/10 text-amber-400 border border-amber-500/20 rounded-lg px-3 py-2 text-xs
"Re-publish your site for SEO changes to take effect."
```

### 2. OG Preview Card

**Purpose:** Shows users how their link will appear when shared on social media.

**Dimensions:** `w-full`, border, rounded-xl, bg-surface-elevated

**Structure:**
```
┌─────────────────────────────────┐
│ [Cover image 16:9 or placeholder] │
├─────────────────────────────────┤
│ Site Title (truncated to 1 line) │
│ site.com/host/username          │
│ Description text truncated...    │
│                          [OG] [X] │
└─────────────────────────────────┘
```

**OG badge:** `bg-accent/20 text-accent text-xs px-2 py-0.5 rounded font-medium`
**Twitter badge:** `bg-surface text-text-muted text-xs px-2 py-0.5 rounded`

**Placeholder image:** Use a gray oklch box with a camera icon when no cover image is set.

### 3. SEO Toolbar Button

**Location:** EditorToolbar.tsx, right side, before the Publish button, separated by vertical divider.

**Icon:** Inline SVG of a magnifier with sparkle or a chain-link icon (represents SEO/social sharing).

**States:**
- Default: `text-text-secondary hover:bg-surface-elevated hover:text-text-primary`
- Panel open: `text-primary bg-primary/10`

## Interaction Behaviors

### Opening SEO Panel
1. User clicks SEO icon in toolbar
2. Panel slides in from right (300ms ease-out-quart)
3. Backdrop is NOT used (panel overlays canvas, editor remains interactive behind)
4. Clicking outside panel does NOT close it (intentional - prevents accidental dismissal)
5. Only close button or pressing Escape closes panel

### Saving SEO Settings
1. User edits meta title/description
2. Changes auto-save after 500ms debounce (same as existing auto-save pattern)
3. For robots.txt: explicit Save button required (changes not auto-saved)
4. On save success: no explicit success message (panel stays open, data persisted)
5. On save failure: show inline error below the field, red border on input

### Re-publish Warning
- Amber info notice visible at all times in the Meta Tags tab
- Notice text: "Re-publish your site for SEO changes to take effect."
- This is informational only, no action required to dismiss

### Tab Navigation
- Two tabs: "Meta Tags" and "robots.txt"
- Active tab: `text-primary border-b-2 border-primary`
- Inactive tab: `text-text-muted hover:text-text-secondary`
- Tab panel switches content without animation (instant swap)
