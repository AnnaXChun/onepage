# Phase 18 Plan 01: UI Foundation

## Objective

Establish white/light theme foundation with blue-black accents across all pages. Implement consistent button hover animations and refine template gallery styling.

## Tasks

### Task 1: Update global CSS variables for white theme

**read_first:**
- `frontend/src/global.css`
- `frontend/tailwind.config.js`

**acceptance_criteria:**
- [ ] `bg-background` set to `#ffffff` (not dark)
- [ ] `text-primary` set to `#1e293b` (dark slate)
- [ ] `text-secondary` set to `#64748b` (medium slate)
- [ ] `border-border` set to `#e2e8f0` (light gray)
- [ ] `bg-surface` set to `#f8fafc` (very light gray)
- [ ] Primary accent color `primary` set to `#2563eb` (blue-600)
- [ ] Secondary accent `accent` set to `#1e3a5f` (navy)

**action:**
Update global.css CSS variables to white theme palette:
```css
:root {
  --color-background: #ffffff;
  --color-primary: #2563eb;
  --color-accent: #1e3a5f;
  --color-surface: #f8fafc;
  --color-border: #e2e8f0;
  --color-text-primary: #1e293b;
  --color-text-secondary: #64748b;
}
```

### Task 2: Add button hover animation system

**read_first:**
- `frontend/src/global.css`

**acceptance_criteria:**
- [ ] `.btn-hover` class exists with scale and shadow transition
- [ ] `hover:scale-[1.02]` applied to buttons
- [ ] `hover:shadow-lg` applied to buttons
- [ ] Transition duration `duration-200` set

**action:**
Add button hover utility classes to global.css:
```css
/* Button hover animations */
.btn-hover {
  transition: transform 200ms ease-out, box-shadow 200ms ease-out, background-color 200ms ease-out;
}
.btn-hover:hover {
  transform: scale(1.02);
  box-shadow: 0 10px 15px -3px rgb(0 0 0 / 0.1);
}
```

### Task 3: Update all button components with hover animation

**read_first:**
- `frontend/src/components/Header/Header.tsx`
- `frontend/src/pages/Home/Home.tsx`
- `frontend/src/pages/Templates/Templates.jsx`

**acceptance_criteria:**
- [ ] All `<button>` elements have `btn-hover` class
- [ ] All primary buttons have `hover:bg-blue-700` for blue buttons
- [ ] Navigation links have `hover:text-primary` transition

**action:**
For each button in Header, Home, and Templates:
1. Add `className="... btn-hover"` to primary action buttons
2. Add `className="... hover:bg-slate-100"` to secondary buttons

### Task 4: Polish template gallery cards

**read_first:**

2. Template gallery cards have refined shadows and smooth hover
