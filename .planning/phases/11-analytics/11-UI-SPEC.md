## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | oklch(15% 0.015 260) | Page background |
| Secondary (30%) | oklch(20% 0.015 260) | Cards, stat containers |

| Destructive | oklch(60% 0.16 25) | Error states only |

**Accent reserved for:** Large stat numbers (visitor count, page views), trend up arrows, positive change indicators

**Background usage:**
- `bg-background` (oklch 15%) — page background
- `bg-surface` (oklch 20%) — cards, stat containers, table rows
- `bg-surface-elevated` (oklch 25%) — hover states, selected items

**Text usage:**
- `text-text-primary` (oklch 98%) — headings, primary numbers
- `text-text-secondary` (oklch 75%) — labels, descriptions
- `text-text-muted` (oklch 55%) — timestamps, secondary info

## Component Inventory

### StatCard

**Purpose:** Display a single metric (visitor count or page views)

**States:**
- Default: Surface background, large accent-colored number, label below
- Loading: Skeleton pulse animation (opacity 50% cycling)
- Error: Muted text "—" instead of number

**Visual spec:**
- Container: `bg-surface rounded-2xl p-6 min-h-[120px]`
- Number: `text-4xl font-semibold text-accent`
- Label: `text-sm text-text-secondary mt-1`
- Trend indicator: `inline-flex items-center gap-1 text-xs` (green for up, red for down)

### AnalyticsTable

**Purpose:** List all user blogs with their analytics

**Visual spec:**
- Container: `w-full bg-surface rounded-2xl overflow-hidden`
- Header row: `bg-surface-elevated px-6 py-4 text-left text-sm font-medium text-text-secondary`
- Data row: `px-6 py-4 border-t border-border`
- Hover: `hover:bg-background transition-colors duration-200`
- Blog name: `font-medium text-text-primary`
- Stats: `text-right font-mono text-sm`
- View button: `px-4 py-2 bg-primary text-white rounded-lg text-sm font-medium`

### PeriodSelector

**Purpose:** Switch between time periods (7 days, 30 days, 90 days)

**Visual spec:**
- Container: `inline-flex bg-surface rounded-lg p-1`
- Option: `px-4 py-2 rounded-md text-sm font-medium transition-all duration-200`
- Active: `bg-primary text-white`
- Inactive: `text-text-secondary hover:text-text-primary`

### EmptyState

**Purpose:** Shown when user has no published blogs or no analytics data

**Visual spec:**
- Container: `flex flex-col items-center justify-center py-16 text-center`
- Icon: `w-16 h-16 text-text-muted mb-4` (chart icon)
- Heading: `text-xl font-semibold text-text-primary mb-2`
- Body: `text-text-secondary mb-6 max-w-sm`

## Interaction Patterns

### Dashboard Load
1. Show loading skeletons for StatCards (100ms stagger)
2. Fetch analytics data from API
3. Fade in stats with slide-up animation (300ms ease-out-quart)
4. If error, show inline error message with retry button

### Period Switch
1. User clicks period option
2. Immediately update active state (no loading needed — data pre-fetched)
3. Re-render stat cards with new aggregated data
4. Fade transition between values (150ms)

### Table Row Hover
1. Background shifts to `background` color
2. View Analytics button becomes visible (opacity 0 to 1, 150ms)
