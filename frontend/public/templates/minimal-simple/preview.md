# Minimal Simple 模板预览

```
┌─────────────────────────────────────┐
│  ┌─────────┐                         │
│  │   图片   │    标题区域              │
│  │ (圆形)  │    博客内容...           │
│  └─────────┘                         │
│                                     │
│   用户名                            │
│   用户简介                          │
│─────────────────────────────────────│
│   博客正文内容区域                   │
│   简洁优雅的文字排版                 │
│                                     │
└─────────────────────────────────────┘
```

特点:
- 左侧固定侧边栏设计
- 圆形头像展示
- 淡入动画效果
- 响应式布局适配移动端
