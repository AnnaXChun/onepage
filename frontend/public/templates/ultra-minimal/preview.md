# Ultra Minimal 模板预览

```
┌───────────────────────────────────┐
│                                   │
│          ╭────╮                   │
│          │头像│   USERNAME        │
│          ╰────╯   bio text...     │
│                                   │
│ ─────────────────────────────    │
│                                   │
│   Post Title                      │
│   Post content goes here...       │
│                                   │
│   [link]                          │
│                                   │
│   Another Post                    │
│   More content...                 │
│                                   │
└───────────────────────────────────┘
```

特点:
- 极致留白设计
- 黑白单色配色
- 小号头像
- 无装饰元素
- 纯文本导向
- 选中文字高亮效果
- 淡入动画
